# otus-calculator

ДЗ - консольный калькулятор

Calculator with tests

Available operations:

- \+ - Addition - "1 + 2"
- \- - Subtraction - "1 - 2"
- \* - Multiplication - "1 \* 2"
- / - Division - "1 / 2"
- ! - Factorial - "2 !"
- \*\* - Exponentiation with power "2 - 2 \*\*"
- ^ - Exponentiation - "1 ^ 2"

Examples:

- 7 + 2 \* 5 = 17
- 9 / 3 - 5 = -2
- 5 \* 2 \*\* - 11 = 9
- 3 \* 3 ! = 18
- 3 \* 3 ^ 4 = 243

Operation priorities:

1. !, \*\*, ^
2. \*, /
3. +, -
